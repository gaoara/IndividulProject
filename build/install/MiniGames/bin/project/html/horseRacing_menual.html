게임의 법칙

1. 간단 요약

- 사기경마는 순위와 관계없이 1등말과 2등말을 맞추면 됩니다.

- 하지만 사기경마는 일반적인 경마와는 다르게 사전에 순위가 정해져있다.

- 플레이어는 게임을 시작하기 전 말 순위와 관련된 단서를 받는다.

- 20개의 칩이 주어진다.

- X라운드마다 칩 0~3개 걸 수있다.

- 베팅 불가능지역에 들어가면 베팅 할 수없다.

- 자신의 베팅한 말에 칩의 개수가 많을수록 배당률은 떨어지고, 칩의 개수가 적을수록 배당률은 올라간다.

- 게임종료 후 베팅에 사용하지않은 칩은 소거된다.

- 1등 2등 말에 한에서 획득한 베팅 칩 수에 따라 최종순위가 결정된다.

2. 추가된 법칙

- 라운드마다 순위와 관련된 단서를 받는다.

3. 삭제된 법칙

- 추가 단서는 가넷 3개로 추가 구입할 수 있다.

- 게임 종료 후, 칩의 개수 10개마다 가넷 1개로 교환할 수 있다.

4. 세줄 요약

(1). 플레이어는 단서를 라운드마다 단서를 받는다.

(2). 1등, 2등말을 맞히면 배당률에 따라 칩을 받는다.

(3). 칩의 개수에 따라 최종순위가 결정된다.
